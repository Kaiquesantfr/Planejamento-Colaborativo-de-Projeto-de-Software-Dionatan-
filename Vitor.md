## SISTEMA DE USUÁRIO 

* 1.Criar tela inicial
Descrição: Desenvolvimento da tela principal do aplicativo com apresentação do sistema.

* 2.Criar cadastro de usuário
Descrição: Criação de formulário para registro de novos usuários.

* 3.Criar login
Descrição: Implementação da tela de login com autenticação básica.

* 4.Criar logout
Descrição: Função para encerrar a sessão do usuário.

* 5.Criar perfil do usuário
Descrição: Tela onde o usuário pode visualizar e editar suas informações.

## SISTEMA DE ROUPAS

* 6.Adicionar roupa
Descrição: Funcionalidade para cadastrar uma nova peça de roupa.

* 7.Remover roupa
Descrição: Permitir a exclusão de roupas cadastradas.

* 8.Editar roupa
Descrição: Permitir alterar informações de uma roupa já cadastrada.

* 9.Categorizar roupa
Descrição: Classificar roupas por tipo (camiseta, calça, etc).

* 10.Adicionar imagem da roupa
Descrição: Permitir anexar uma imagem para cada peça cadastrada.

## SISTEMA DE OUTFITS

* 11.Criar outfit
Descrição: Permitir montar combinações de roupas.

* 12.Editar outfit
Descrição: Alterar combinações já criadas.

* 13.Excluir outfit
Descrição: Remover outfits salvos.

* 14.Visualizar outfits salvos
Descrição: Listar todas as combinações criadas pelo usuário.

* 15.Compartilhar outfit
Descrição: Permitir compartilhar outfits com outros usuários.
